= Drop Tweet

A simple project to create image tweet.

== Getting Started

1. <tt>git clone git://github.com/chawei/droptweet.git</tt>
2. <tt>sudo gem install bundler</tt>
3. <tt>bundle install</tt>
4. <tt>rackup</tt>
